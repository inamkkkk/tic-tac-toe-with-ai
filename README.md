# Tic-Tac-Toe with AI

A simple Tic-Tac-Toe game implemented in Flutter with a basic AI opponent.

## Features

*   Classic Tic-Tac-Toe gameplay.
*   AI opponent using a simple algorithm.

## Getting Started

1.  Clone the repository.
2.  Run `flutter pub get` to install dependencies.
3.  Run `flutter run` to start the game.
